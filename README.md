# 雅思打怪兽背单词 · Word Monster

小女孩打怪兽背单词小游戏：每只怪兽代表一个单词，用三种攻击模式（英文选中文、中文选英文、Spell it）击败怪兽，帮助记忆雅思词汇。

- **词库**：来自同目录下雅思词汇 PDF 提取的 `public/words.json`（约 1400 词），可按需替换。
- **部署**：适用于 Vercel 静态部署。

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开 [http://localhost:5173](http://localhost:5173)

## ·构建与部署到 Vercel

```bash
npm run build
```

将本文件夹通过 Git 推送到 GitHub，在 Vercel 中导入该仓库，框架选 **Vite**，构建命令与输出目录会自动识别（或使用根目录 `vercel.json`）。部署后访问站点即可玩。

## 游戏规则

- 每只怪兽 = 一个单词，答对即击败，进入下一只。
- 三种随机题型：
  - **英文选中文**：看到英文，从四个选项中选正确中文。
  - **中文选英文**：看到中文，从四个选项中选正确英文。
  - **Spell it**：根据中文释义，在输入框拼写英文单词（回车或点击「攻击」提交）。
- 生命值 3 点，答错扣 1 点；扣完即本局结束。可重复挑战同一只怪兽直到答对或生命耗尽。
- 通关全部词库或生命耗尽后显示本局击败数量，可「再玩一局」或返回首页。
- **错题记录**：每个单词答错一次（含第一次提示、第二次扣血）都会累计到该词的错题次数；错题记录**不会**随「重置通关进度」清除。
- **错题模式**：首页可选「错题模式（只出错过的词）」；仅当至少有一个词曾答错过时可用，专门练习错题。
